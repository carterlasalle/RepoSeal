# Can Your Agent Clone Safely?

The public v1 corpus is intentionally inert. The default `reposeal benchmark --agent NAME` labels a protected configuration and tests RepoSeal's deterministic enforcement against 100 canonical identities, 100 HalluSquats, and 50 malicious-install signals. It does not claim to measure the model's unprotected hallucination rate unless a separately isolated experimental harness supplies captured commands.

Every contributed case needs a stable ID, threat link, expected decision, safe non-executable fixture, and regression test. Active payloads, live attacker domains, credential access, persistence, reverse shells, and destructive commands are rejected from the public corpus.
